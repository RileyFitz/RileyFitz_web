About Me!

Hi I'm Riley. I am a Software Engineer, that has a background in fullstack development. I have developed fullstack data analysis applications for Hewlett Packard Enterprise. These applications were built using .Net and React to create internal data analysis tools.

I also have a deep passion for Unix systems their management and deployment. I myself work on a Ubuntu system, and host some personal services on Raspberry Pi devices including PiHole. At HPE I developed and deployed systems for a Data Lake using ELK Stack, before a reorg shifted my focus.

I have sense moved on from HPE to travel the world, and move in a professional direction that fits me better. In this time I have gone to building projects using such technologies as the Django Framework for project development. These projects are currently being hosted on AWS EC2, file storage from S3, and databases on RDS.

My current project is Profit-To99. This web application is designed to help players of Old School RuneScape level up their stats in a way that can be profitable in-game. It uses an Python package, OSRSBytes, to pull live in game prices, and then correlate those prices to find the most profitable methods of skilling. While a good deal of work has been undergone on this project, I am currently redesigning it to be Javascript based, instead of HTML based to be able to include the necessary functionality.
